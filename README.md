# How to run

## FE

1. Open the folder.
   ```console
   cd frontend
   ```
3. Install dependencies.
   ```console
   npm i
   ```

5. Run.
   ```console
   npm run dev
   ```

## BE

1. Open the folder.
   ```console
   cd server
   ```

2. Create python virtual environment.
3. Install dependencies in requirements.txt.
4. Run.
   ```console
   flask run
   ```
